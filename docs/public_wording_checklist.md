# Public Wording Checklist

Use this checklist before updating README, demo, pitch, or reviewer-facing docs.

## Lead With

- Aletheia is a local-first agent harness for sensitive professional document
  work.
- Short analogy: Codex for sensitive professional work.
- Product shape: Aletheia Kernel plus Domain Packs.
- First public/private-pilot pack: Private Contract / Due Diligence Review.

## Kernel Terms

- Local Vault;
- Agent Loop Runtime;
- Typed Artifact Graph;
- Permission + Tool Policy;
- Review + Gate Console;
- Audit Trace;
- Eval Replay;
- Human-approved Skills.

## Domain Pack Terms

- Private Contract / Due Diligence Review Pack;
- Compliance Obligation Pack;
- Audit Evidence Pack;
- Regulatory Response Pack;
- Litigation Chronology Pack.

## Keep Caveats Visible

- Local/private-pilot scope unless stronger validation exists.
- Not production SaaS.
- Not legal advice.
- Not a replacement for lawyers, auditors, compliance officers, or other
  qualified experts.
- External model calls are off by default for sensitive/private data.
- Expert review and gates are required before high-risk final exports.

## Avoid In Public Docs

- Presenting legal/compliance/audit/regulatory as separate top-level product
  categories.
- Calling Aletheia a generic multi-industry SaaS.
- Claiming production readiness, guaranteed correctness, or compliance
  certification without evidence.
- Internal coordination jargon or repo-state handoff language.
